# Modern CSS Resources

## CSS Features

### Modern CSS
https://moderncss.dev/

Modern CSS techniques, best practices, and solutions. Reference for contemporary CSS patterns.

### CSS Logical Properties
https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Logical_Properties

Documentation for logical properties that adapt to writing direction. Use instead of physical properties (left, right, top, bottom).

### CSS Cascade Layers
https://developer.mozilla.org/en-US/docs/Web/CSS/@layer

Specification for `@layer` at-rule. Use to organize CSS in predictable layers.

### OKLCH Color Space
https://developer.mozilla.org/en-US/docs/Web/CSS/color_value/oklch

Documentation for OKLCH color function. Provides perceptually uniform colors with wider gamut.

## User Preferences

### prefers-reduced-motion
https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-reduced-motion

Media query for respecting user motion preferences. Use to disable animations when requested.

### prefers-reduced-transparency
https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-reduced-transparency

Media query for respecting transparency preferences. Use to disable glass/blur effects when requested.

### prefers-color-scheme
https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-color-scheme

Media query for detecting light/dark mode preference. Use with `color-scheme` property.

### forced-colors
https://developer.mozilla.org/en-US/docs/Web/CSS/@media/forced-colors

Media query for Windows High Contrast Mode. Use to ensure UI remains visible.

## Typography

### text-wrap
https://developer.mozilla.org/en-US/docs/Web/CSS/text-wrap

Property for controlling text wrapping behavior. Use `balance` for headings, `pretty` for paragraphs.

### font-feature-settings
https://developer.mozilla.org/en-US/docs/Web/CSS/font-feature-settings

Control OpenType font features. Use `"ss06"` for San Francisco font improvements.

## Tools

### Autoprefixer
https://github.com/postcss/autoprefixer

PostCSS plugin for adding vendor prefixes automatically. Use for cross-browser compatibility.

### Can I Use
https://caniuse.com/

Browser support tables for CSS features. Check before using modern CSS properties.
